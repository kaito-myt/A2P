# 08 — 販促プレイブック（本紹介 SNS の勝ちパターン）

Kindle 実用書/ビジネス書/自己啓発の SNS 販促（Instagram / X / TikTok）で「売れる/伸びる」投稿の
実装可能ルール集。web リサーチ（2026-07）に基づく。**販促画像コンポーザ・戦略プレイブックエージェント・
TikTok 台本ジェネレータの設計根拠**として使う。関連: [[project_promotion_features]]、docs/05 販促付録。

## 1. IG フィード販促「画像」設計（購買を動かす）

勝ち型は **ベネフィット見出しが主役・表紙は"証拠"**（表紙単体では売れない）。コンポーザ実装ルール:

1. **アスペクト比 4:5（1080×1350）**。上下 ~120px はセーフゾーン（UI 被り）＝テキスト置かない。
2. **見出し＝最大要素**。ベネフィット/好奇心ギャップ1行、1行13–20字・最大2行、上1/3を占有、bold/black 90–140px。
3. **数字・具体で**。「3つの習慣」「月5万円」「1日10分」。数字はアラビア数字。
4. **アクセント色は1つだけ**。ベース=白/近黒、ペイオフ語だけ蛍光バンド/色文字で強調。強調は2要素まで。
5. **高コントラスト・フォント2種以内**。画像上テキストは半透明スクリムで可読性 4.5:1 以上。
6. **KU バッジ**。対象なら右上にリボン/ピル「Kindle Unlimited 読み放題」「★KU 0円」。ゼロリスク訴求＝強い転換レバー。
7. **表紙サムネは隅**（フレームの30%以下・影付き）で"本物の本"に見せる。
8. **固定テンプレ＝ブランド想起**。色/フォント/ロゴ位置を全投稿で固定。
9. **How-to/リストはカルーセル＞単一**。1枚目=見出しフック、2–N=1要点ずつ、最終=表紙+CTA。5–8枚。保存/シェアを稼ぐ（2026最重要シグナル）。
10. **ジャンル別パレット**: money=緑/金、health=青/白、self-help=暖色オレンジ、競馬=芝グリーン+赤。per-genre トークン化。

## 2. キャプション（IG / X）

構造 = **Hook → Value → CTA（HVC）**:
- **1行目=結論/約束**（「〜な人が9割。実はこれだけで変わります」）。"…続きを読む"前の1行で決まる。
- **本文**: 価値3点＋補足1。短行・ブロック間に空行。絵文字は行頭マーカー（✅▶️📌）を1行1個まで。
- **長さ**: IG 150–500字（価値を前に）。X ≤140字/tweet、how-to は5–7連スレッド。
- **検索KWを自然文に織り込む**（IG はキャプション本文もインデックス）。
- **CTA**: IG キャプション内リンクは不活性 → bio 誘導「詳細はプロフィールのリンクから📚」。**bio リンク先の先頭=Amazon/KU ページ（or /books ランディング）**。「保存して後で」「コメントで教えて」でシグナル増。X は最後のtweetに Amazon リンク。
- **価値:宣伝 = 8:2**。

## 3. ハッシュタグ

- **数=10–15**（旧30より質）。3層ミックス:
  - **ビッグ(>50万)**: #読書(~162万) #読書記録(~63万) #読了(~24万)
  - **ミッド(5万–30万)**: #読書好きな人と繋がりたい(~25.5万) #本好きな人と繋がりたい(~18.4万) #読書倶楽部 #活字中毒
  - **ニッチ(<5万・ジャンル/意図)**: #ビジネス書 #自己啓発本 #お金の勉強 #副業 #Kindle出版 #kindleunlimited #競馬予想 ← ここが転換する
- **比率 ~ ビッグ2 / ミッド5 / ニッチ5–8**。ニッチのtopに載る狙い。
- **配置**: 先頭コメントに置きキャプションを綺麗に（リーチ同等）。タグセットは回転させる。

## 4. 投稿頻度・時間（JP）

- **IG フィード/カルーセル: 20:00–22:00**、水木寄り。Reels は peak の1–2h前(18–19時)。初速が命。
- **X: 7–8時/12時/21–23時**（夜が最活発）。
- **TikTok: 19:00–23:00**、次点12:00/7:00。
- **頻度**: IG 3–5回/週+毎日stories、X 2–3投稿/日、TikTok 最低1回/日（量が発見を生む）。
- 複利は **~3ヶ月**。初期は実験と割り切る。

## 5. TikTok テンポ（台本ジェネレータ）

- **尺: 21–34s**（1冊フック）。上限45s。
- **フックは ≤1s**（音声＋画面テキスト両方）。最初3sの維持率<70%で失速。フック型:
  - 警告: 「〇〇してる人、今すぐやめて」/ リスト: 「お金が貯まる人の3つの口癖」
  - 好奇心ギャップ: 「この本の"ある1ページ"で人生変わった」/ 逆張り: 「9割が知らない〇〇」
- **構成**: フック(0–1s)→緊張/理由(1–4s)→ペイオフ3ビート→クリフハンガー/CTA。最良点は最後まで引っ張る（リプレイ/コメント誘発）。
- **カット尺: 各1.5–3s、8–12シーン**。静止1フレームを3s超保持しない。
- **画面テキスト**: 各シーンに8–16字、ナレーションに同期、中央セーフゾーン（下15%はUI/字幕で避ける）。
- **ナレーション**: 早口 ~6–7モーラ/秒、無音を作らない。~120–160字/30s。
- **音楽**: 今トレンドの音源（毎週確認）、VO下で小さく。
- **キャプション**: フック1行＋タグ3–5（#本紹介 #読書 #ビジネス書 #kindle）。CTA「続きはプロフから📚」。
- **ジェネレータ既定**: `scenes=8–12`, `sec_per_scene=1.5–3`, `total=21–34s`, `hook_type∈{warning,list,curiosity,contrarian}`, `payoff_beats=3`, `cliffhanger=true`。

## 6. A/B テスト変数と勝ち指標

| 変数 | テストする値 |
|---|---|
| 見出し型 | 数字リスト / 好奇心ギャップ / 警告 / ビフォーアフター |
| 画像レイアウト | テキスト主役 / 表紙主役、単一 / カルーセル |
| アクセント色 | ジャンル別パレット |
| KU バッジ | 有 / 無 |
| フック型(TikTok) | 上記4型 |
| 動画尺 | 15 / 25 / 35s |
| CTA文言 | プロフのリンクから / 保存して後で / コメントで |
| ハッシュタグ | ビッグ寄り / ニッチ寄り |
| 投稿時間 | 20:00 / 21:00 / 12:00 |
| キャプション長 | 短打 / フルHVC |

**勝ちフォーマット指標（優先順）**:
1. **シェア/送信(IG)・完了+再視聴%(TikTok)** ＝ 2026 の発見シグナル最上位
2. **保存**（how-to の購買意図プロキシ）
3. **bio リンクCTR/プロフ訪問**（Amazon クリックに最も近い）
4. リーチ/インプレ → いいね（最弱）

**1変数ずつ**・1アーム 5–7投稿。いいねでなく **シェア+保存+リンクCTR** を上げたアームを採用。
